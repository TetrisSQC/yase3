The story

Growing up in East Germany, a ZX Spectrum was a window into a different world — one that wasn't officially supposed to be accessible. That machine left its mark.

I wrote the first version in 1998, in Turbo Pascal 6 on DOS. Then Delphi 7. Then FireMonkey. Porting all of that to JavaScript would have been enormous — so for the browser version I built on top of JSSpeccy 3, Thomas Harte's WebAssembly Z80 core. That foundation saved years of work.

What YaSE adds is everything JSSpeccy 3 didn't cover: the additional machines, peripherals, file formats, debugger, CRT shader, and PWA layer. With some help from AI, it finally became what I had in mind thirty years ago. It's done.

  • WebGL CRT shader — scanlines, vignette, distortion, flicker, pixel-shift
  • Configurable aspect ratio and filter intensity
  • Touch joystick + on-screen ZX keyboard overlay for mobile
  • Gamepad API → Kempston joystick mapping

Developer tools

  • Live Z80 debugger — disassembler, breakpoints, step / step-over
  • Register dump + full flag display
  • Memory viewer — hex + ASCII, goto, poke
  • Pokefinder for cheat codes
  • RZX input record + deterministic playback
  • Snapshot export (.z80)

For developers

How it works

Open source, thoroughly attributed, built on open standards. Pull requests welcome.

Z80 core AssemblyScript compiled to WebAssembly — near-native Z80 speed in every modern browser. Adapted from JSSpeccy 3 (Thomas Harte) with Fuse 1.8.0 machine-model extensions (Philip Kendall et al.).
Threading Emulation loop runs in a Web Worker. The UI thread is never blocked — stays at a smooth 60 fps regardless of emulation load.
Framebuffer Per-frame postMessage carries the 256×192 pixel framebuffer and AY-3-8912 audio sample block to the UI thread for WebGL rendering.
RZX replay 8192-byte input buffer per frame ensures deterministic replay across all 15 emulated machine models. Record and playback are frame-accurate.
Offline / PWA Cache-first service worker pre-caches ROM files, the Wasm module, and all app assets on first visit. Works completely without a network connection thereafter.
